## About the Build

The goal of this build was to expand my electonics knowledge along with physically creating a project that I would be proud over. This keyboard was designed following the excellent guide by Masterzen, [**Designing a keyboard from scratch**](https://www.masterzen.fr/2020/05/03/designing-a-keyboard-part-1/). 


## Repository Contents:

- KiCad schematic (`.kicad_sch`) and PCB layout (`.kicad_pcb`) files
- Footprint & symbol libraries
- Gerber files
- `images/` — renders and PCB design images

## Tools Used

- **KiCad** — schematic capture and PCB design
- **SolidWorks** — 3D modeling and renders
- **QMK** — keyboard firmware (planned)
